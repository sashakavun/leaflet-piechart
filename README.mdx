# Leaflet Charts

## Pie Charts

Just add the pie chart marker using the basic code:

```javascript
L.piechartMarker(L.latLng([lat, long]), {
  data: [
    { name: "Apples", value: 25 },
    { name: "Oranges", value: 35 },
    { name: "Bananas", value: 20 },
    { name: "Pineapples", value: 30 },
  ],
}).addTo(map);
```
